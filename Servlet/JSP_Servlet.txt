jsp 서블릿

클라이언트와 서버의 관계를 먼저 이해해야함

클라이언트가 서버에다 request (http형태로)
그러면 서버는 클라이언트에다가 response (html형태로 (도큐먼트로 구조화시켜서)





클라이언트가 요청해주면 서버가 받아서 데이터베이스에서 값가져올거 있으면 가지고와서
html을 만들어서 응답해준다.



자바가 서버 DB는 말그대로 데이터 모아둔곳
UI가 클라이언트가 될거임 
서블릿은 제어해주는역할( 내가 뭐 보여줘라고 서버에서 응답해줄걸 제어해줌 )

JSP
(java server page)

html안에 <% 자바명령어 %> 이것이 jsp

자바안에 html만들어주는게 서블릿
자바안에서 html만들어서 하나하나 만들어서 응답해주려니깐 너무빡세서

jsp를 만듬( 동적 )
